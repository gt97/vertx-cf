Demo pipeline per ciclo di vita applicazioni cloud-native

Abbiamo realizzato un paio di servizi di esempio per dimostrare come si possa gestire l'intero ciclo di vita di una applicazione collaborativamente sul cloud, nel nostro caso i principali passaggi di tale ciclo sono:

  1. stesura dei requisiti tecnici/funzionali o segnalazione di bug
  2. scrittura del codice per soddisfare il requisito
  3. versionamento del codice sorgenge
  4. discussione e processo di accettazione del codice
  5. build e test unitario completamente automatizzato ,tracciato e fedelmente riproducibile
  6. rilascio automatizzato in ambiente di stage per il test di accettazione
  7. processo decisioname per promuove il change al passaggio in produzione o per il rollback dell'ambiete di stage
  8. eventuale rilascio, automatizzato, tracciato e rollbackabile in ambiente di run
